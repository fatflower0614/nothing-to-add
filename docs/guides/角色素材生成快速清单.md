## ✅ 一致性检查（每生成一张图）

- [ ] 面部轮廓一致
- [ ] 眼镜/配饰位置相同
- [ ] 发型/发色一致
- [ ] 服装款式和颜色统一
- [ ] 整体风格匹配
- [ ] 光照方向一致

**如果不一致**:
1. 降低相似度阈值到80%
2. 重新生成
3. 或用GIMP手动调整

### 混元3.0 - 巴菲特基础

卡通风格的沃伦·巴菲特头像，中年白人男性，戴黑色边框眼镜，穿西装打领带，表情平和友善，风格简洁扁平，白色背景，High quality, cartoon style, vector art

```

### 海螺AI - 动作帧（使用主体参考）

```

## 🆘 遇到问题

### 生成质量不佳
→ 调整提示词，添加"专业插画师级别"

### 一致性差
→ 降低相似度阈值到80%，或手动用GIMP调整

### 动画不流畅
→ 增加帧数到8-10帧

### 文件太大
→ 用TinyPNG压缩: https://tinypng.com/
