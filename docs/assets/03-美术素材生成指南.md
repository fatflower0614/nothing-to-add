## 1. 美术风格定义

### 1.1 核心风格：美漫风格

**参考作品**：
- 《穷查理宝典》插画
- 《Rick and Morty》动画风格
- 经典美式漫画

**风格特征**：
```
✓ 清晰的黑色轮廓线
✓ 平涂色彩（flat coloring）
✓ 鲜明饱和的色彩
✓ 夸张但 recognizable 的面部特征
✓ 简洁有力的线条
✓ 表情丰富
✓ 动态感强
```

### 1.2 色彩方案

**巴菲特配色**：
- 主色：橙色（#F97316）
- 辅助色：金色（#FBBF24）
- 背景：暖色调渐变

**芒格配色**：
- 主色：蓝色（#3B82F6）
- 辅助色：银色（#9CA3AF）
- 背景：冷色调渐变

### 1.3 线条风格

- **轮廓线**：黑色，粗细一致
- **阴影**：简化，用纯色块
- **高光**：用亮色点缀

## 5. 技术规格

### 5.1 文件规格

| 类型 | 格式 | 尺寸 | DPI | 背景 |
|------|------|------|-----|------|
| **角色基础图** | PNG | 1024×1024 | 72 | 透明 |
| **表情图** | PNG | 1024×1024 | 72 | 透明 |
| **嘴巴帧** | PNG | 512×512 | 72 | 透明 |
| **场景背景** | JPG | 1920×1080 | 72 | 不透明 |

### 5.2 命名规范

**角色**：
```
{角色名}-{类型}-{细节}.png

例如：
buffett-body-sitting.png
buffett-face-smile.png
buffett-mouth-01.png
munger-face-serious.png
```

**场景**：
```
bg-{场景名}.jpg

例如：
bg-shareholders.jpg
bg-lunch.jpg
```

## 6. 生成流程

### 6.1 第一阶段：角色基础（1-2天）

**步骤1**：生成巴菲特基础图
1. 打开可灵AI
2. 输入提示词
3. 生成10-20张
4. 选择最像的3-5张
5. 保存为 `buffett-body-sitting.png`

**步骤2**：生成芒格基础图
1. 相同流程
2. 保存为 `munger-body-sitting.png`

---

### 6.2 第二阶段：表情变化（1-2天）

**步骤1**：巴菲特表情
1. 基于基础图，修改提示词
2. 生成每个表情（7个）
3. 选择最像的
4. 保存

**步骤2**：芒格表情
1. 相同流程
2. 生成每个表情（7个）
3. 保存

---

### 6.3 第三阶段：场景背景（1天）

**步骤**：
1. 为每个场景生成提示词
2. 生成背景图
3. 选择最符合风格的
4. 调整色彩（可选）
5. 保存

---

### 6.4 第四阶段：嘴巴动画（可选，半天）

**如果用CSS**：不需要生成

**如果用Sprite**：
1. 生成10-20帧渐进图
2. 确保一致性
3. 保存为序列

## 8. 实用技巧

### 8.1 提高生成质量

**技巧1：使用参考图**
- 上传《穷查理宝典》的插画作为参考
- 可以大幅提升一致性

**技巧2：固定Seed**
```
可灵AI可能不支持seed
Midjourney：--seed 12345
```

**技巧3：迭代优化**
```
第一轮：生成基础图
第二轮："基于上一张，让表情更温暖"
第三轮："稍微调整嘴巴"
```

**技巧4：批量生成**
- 一次生成10-20张
- 从中挑选最好的
- 不要追求完美

---

### 8.2 后期处理

**必要的后期**：
- 裁剪到正确尺寸
- 调整亮度/对比度
- 统一色调
- 去除杂点

**工具推荐**：
- 在线工具：Remove.bg（去背景）
- 本地工具：Photoshop、GIMP
- 简单工具：Windows自带照片编辑器

---

### 8.3 保持一致性

**方法1：使用相同的提示词模板**

**方法2：后期统一调色**
```
所有图片应用相同的滤镜：
- 饱和度：+10%
- 对比度：+5%
- 亮度：0
```

**方法3：参考同一张作品**
- 始终参考《穷查理宝典》
- 保持线条粗细一致

## 9. 常见问题

### Q1：生成的图片不太像怎么办？

**A**：
1. 添加更具体的描述
2. 使用参考图
3. 多生成几张，挑选最像的
4. 调整提示词，强调关键特征

### Q2：风格不一致怎么办？

**A**：
1. 使用相同的提示词模板
2. 后期统一调色
3. 参考同一作品
4. 固定生成参数

### Q3：背景去除不干净怎么办？

**A**：
1. 使用Remove.bg
2. 用魔棒工具手动调整
3. 要求AI生成透明背景

### Q4：生成次数不够怎么办？

**A**：
1. 多注册几个账号
2. 使用多个工具
3. 隔天再生成
4. 付费购买次数
