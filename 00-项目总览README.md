## 🚀 当前阶段状态

### 开发阶段：Demo/MVP

**选型原则**：
1. ✅ 优先使用免费工具
2. ✅ 优先使用已订阅服务（沉没成本）
3. ✅ 在保证基本功能的前提下最小化开支

- **语音**：Edge-TTS免费
- **转录**：Whisper免费
- **部署**：HuggingFace Spaces免费

**成本**：**$0/年** ✅

## 🛠️ 技术架构概览

```
用户输入（问题）
    ↓
Web界面（Gradio）
    ↓
智能路由（判断任务复杂度）
    ↓
┌─────────────┬─────────────────┐
│  简单任务   │    复杂任务     │
│ HuggingFace │   GLM-4.7       │
│   免费 API  │   (已订阅)      │
└─────────────┴─────────────────┘
    ↓             ↓
RAG系统（检索巴菲特/芒格的真实内容）
    ↓
生成回答（保持风格一致性）
    ↓
返回用户（带引用来源）
```
