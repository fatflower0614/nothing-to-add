# Nothing to Add Frontend

巴菲特与芒格AI Agent的前端应用

## 技术栈

- React 18
- Vite
- TailwindCSS
- Axios

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 功能特性

- ✅ 美观的聊天界面
- ✅ 巴菲特和芒格动画头像
- ✅ 实时对话
- ✅ 语音播放回答
- ✅ 快捷问题
- ✅ 联网搜索开关
- ✅ 响应式设计

## API配置

默认连接到 `http://localhost:8000`

可通过环境变量修改：
```bash
VITE_API_URL=http://your-api-url
```
